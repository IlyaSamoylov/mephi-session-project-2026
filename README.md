# Развертывание веб-сервера nginx с настройкой сети, хранения и механизмов безопасности Linux

В данном репозитории представлены результаты лабораторной работы по настройке Linux-системы.

В рамках проекта выполнена комплексная настройка Linux-системы с развертыванием веб-сервера `nginx`, организацией отдельного раздела для размещения веб-контента и настройкой механизмов безопасности (SELinux, Linux capabilities, PAM).

Конфигурация выполнена вручную через CLI. Полная последовательность команд зафиксирована в `project_history.txt`.


---
## Сеть

Настроен статический IPv4-адрес с использованием `nmcli`:

- IP-адрес: **192.168.0.150/24**
- Шлюз по умолчанию: **192.168.0.1**
- DNS-сервер: **8.8.8.8**
- имя хоста mephi-2026.domain.local

Проверка маршрутизации и доступности сети выполнена с помощью `ip route` и `ping`.

## Хранение данных

- Создан раздел на дополнительном диске `/dev/sdb`
- Файловая система отформатирована в `ext4` с меткой `MEPHI_DATA`
- Настроено постоянное монтирование через `/etc/fstab` (по LABEL)
- Точка монтирования: `/data/mephi-web`

Веб-контент размещается на отдельном разделе, независимом от от системного.

## Веб-сервер

- Установлен и включён `nginx`
- Сервер настроен на автозапуск 
- Изменён `root` веб-сервера на `/data/mephi-web`
- Создан тестовый `index.html`
- Доступ проверен локально и по IP-адресу `192.168.0.150`

## Безопасность и доступ

- Создана группа `mephi-devs` и пользователь `mephi-admin` 
- Каталог веб-контента передан пользователю `mephi-admin` и группе `mephi-devs`
- Права каталога настроены как `2775`, установлен setgid бит

Обеспечено групповое управление файлами веб-каталога

### SELinux

- Включён режим `Enforcing`
- Добавлено правило контекста `httpd_sys_content_t` для веб-каталога

### Capabilities
Для `tcpdump`:
- Удалён SUID-бит у `tcpdump`
- Назначены Linux capabilities `cap_net_raw`, `cap_net_admin`

### Ограничение доступа root

- Реализован deny-list для запрещённых пользователей
- Настроено ограничение входа пользователя `root` через PAM

## Артефакты выполнения

Файлы с результатами проверок:

- `fstab.txt` - содержимое etc/fstab
- `selinux_status.txt` - статус SELinux
- `file_contexts.txt` - SELinux-контекст директории `/data/mephi-web` 
- `tcpdump_capabilities.txt` - capabilities, назначенные `/usr/sbin/tcpdump`
- `permissions.txt` - права доступа, владелец, группа и параметры `/data/mephi-web`
- `users_groups.txt` - информация о пользователе `mephi-admin` и группе `mephi-devs`
- `curl_output.txt` - ответ веб-сервера, `Hello from Student: 2551096`
- `project_history.txt` - история выполнения команд
- `index.html` - веб-страница
- `mephi-nginx-screenshot.png` - подтверждение работы

Эти файлы подтверждают состояние системы после выполнения всех этапов работы.
